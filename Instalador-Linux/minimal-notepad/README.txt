Instrucciones de instalación:

1. Abre una terminal

2. Navega al directorio donde está el script:
   cd ~/minimal-notepad

3. Da permisos de ejecución al script:
   chmod +x install.sh

4. Ejecuta el script con sudo:
   sudo ./install.sh

5. Una vez instalado, puedes ejecutar el editor de dos formas:
   - Desde el menú de aplicaciones buscando "Minimal Notepad"
   - Desde la terminal escribiendo: minimal-notepad

Si encuentras algún error durante la instalación, asegúrate de que:
- Tienes una conexión a internet activa
- Estás ejecutando el script con sudo
- Tu sistema está actualizado
